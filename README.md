# QuizMate

QuizMate는 강의자료를 바탕으로 AI 학습 정리본, 대화형 복습 퀴즈, 퀴즈 결과 분석을 지원하는 웹 프로토타입입니다.

## 개발 목적

시험 공부 과정에서 학생들은 강의 내용을 읽기만 해서는 자신이 제대로 이해했는지 확인하기 어렵습니다.  
또한 직접 복습 문제를 만들고, 자신이 어떤 난이도와 문제 유형에서 약한지 분석하는 데에도 시간이 많이 걸립니다.

QuizMate는 이러한 문제를 해결하기 위해 개발한 AI 활용 학습 보조 도구입니다.  
사용자는 강의자료 PDF 또는 텍스트를 입력하고, 원하는 난이도와 문제 유형을 선택해 GPT에 입력할 학습 프롬프트를 생성할 수 있습니다.

## 주요 기능

- 강의자료 PDF 업로드
- txt, md 파일 업로드
- 강의 내용 직접 입력
- 강의자료 기반 PDF 학습 정리본 생성 요청
- 대화형 퀴즈 프롬프트 생성
- 정답을 바로 보여주지 않는 퀴즈 진행 방식
- 사용자의 답변 이후 정답/오답 판단 및 해설 제공
- `정리` 명령어를 통한 오답노트 PDF 생성 요청
- GPT 결과 PDF 업로드 기반 결과 분석
- 난이도별 오답률 그래프
- 문제 유형별 오답률 그래프
- 취약 개념 기반 학습 방향 추천
- CSV 다운로드 기능
- Orange3 또는 Python 분석을 위한 데이터 내보내기
