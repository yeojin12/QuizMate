# 📚 QuizMate

> 강의자료를 기반으로 **AI 학습 정리본**, **대화형 복습 퀴즈**, **퀴즈 결과 분석**을 지원하는 웹 프로토타입입니다.

QuizMate는 시험 공부 과정에서 학생들이 강의자료를 단순히 읽는 데 그치지 않고,  
AI가 생성한 문제를 풀며 자신의 이해도를 점검하고, 퀴즈 결과를 분석하여 다음 학습 방향을 찾을 수 있도록 돕는 학습 보조 도구입니다.

## ✨ Project Overview

QuizMate는 강의자료 PDF 또는 텍스트를 입력하면, 사용자가 선택한 난이도와 문제 유형에 맞춰 GPT에 입력할 학습 프롬프트를 생성합니다.

생성된 프롬프트를 GPT에 입력하면 다음 흐름으로 학습을 진행할 수 있습니다.

```text
강의자료 입력
→ AI 학습 정리본 생성
→ 대화형 복습 퀴즈 진행
→ 오답노트 및 결과 데이터 생성
→ QuizMate 분석 페이지에서 시각화
→ 다음 학습 방향 추천

🎯 Development Purpose
시험 공부 과정에서 학생들은 강의 내용을 읽기만 해서는 자신이 제대로 이해했는지 확인하기 어렵습니다.
또한 복습 문제를 직접 만들고, 자신이 어떤 난이도와 문제 유형에서 약한지 분석하는 데에도 시간이 많이 걸립니다.
QuizMate는 이러한 문제를 해결하기 위해 개발한 생성형 AI 기반 학습 보조 도구입니다.

🔄 Development Process
QuizMate는 한 번에 완성된 도구가 아니라, 실제 사용 과정에서 발견한 문제를 바탕으로 점진적으로 개선되었습니다.
단계	구현 내용	발견한 문제	개선 방향
1차	텍스트 입력 기반 프롬프트 생성	긴 강의 내용을 직접 입력해야 함	PDF 업로드 기능 추가
2차	PDF 파일 업로드	강의자료 활용성 개선	텍스트 추출 기능 적용
3차	문제·정답·해설 동시 생성	문제를 풀기 전에 정답이 보임	대화형 퀴즈 방식으로 개선
4차	PDF 정리본 및 오답노트 생성 요청	학습 후 정리 필요	정리 명령어 추가
5차	JSON 기반 결과 분석 시도	GPT 출력 형식이 자주 깨짐	CSV 기반 분석으로 단순화
6차	QuizMate 분석 페이지 구현	학습 결과를 한눈에 보기 어려움	그래프와 추천 기능 추가
7차	Orange3 연계	외부 분석 가능성 확인	CSV 내보내기 기능 활용
🧪 Trial and Error
초기에는 GPT가 퀴즈 결과를 JSON 형식으로 출력하도록 설계했습니다.
하지만 실제 테스트 과정에서 긴 문장, 줄바꿈, 따옴표 등이 포함되면서 JSON 구조가 자주 깨지는 문제가 발생했습니다.
이를 해결하기 위해 분석용 데이터 형식을 단순한 CSV 형태로 변경했습니다.
CSV에는 문제 전문과 긴 해설을 제외하고, 세션, 문제 번호, 난이도, 문제 유형, 정답 여부, 취약 개념, score만 포함하도록 구성했습니다.
이 과정을 통해 AI가 항상 완벽한 형식의 결과물을 제공하는 것은 아니며, 사용자의 의도와 피드백에 따라 구조를 단순화하고 개선해야 한다는 점을 확인했습니다.

🚀 How to Use
1. QuizMate 실행
index.html을 브라우저에서 실행합니다.
또는 GitHub Pages 배포 링크로 접속합니다.
https://깃허브아이디.github.io/QuizMate/
2. 강의자료 입력
PDF 파일 업로드
txt, md 파일 업로드
직접 텍스트 입력
3. 학습 조건 설정
학습 흐름 선택
난이도 선택
문제 수 입력
문제 유형 선택
4. GPT 프롬프트 생성
학습 프롬프트 생성하기 버튼을 누른 뒤, 생성된 프롬프트를 GPT에 붙여넣습니다.
5. 퀴즈 진행
GPT가 한 문제씩 출제합니다.
사용자가 답을 입력하면 GPT가 정답 여부와 해설을 제공합니다.
6. 결과 분석
퀴즈 결과 CSV 데이터를 analysis.html에 업로드하면 QuizMate가 분석을 수행합니다.
📌 Limitations
현재 버전에는 다음과 같은 한계가 있습니다.
GPT API를 직접 연동하지 않아 프롬프트를 복사해 사용해야 함
GPT 결과를 다시 CSV로 정리해야 하는 번거로움이 있음
PDF는 텍스트가 선택되는 자료만 인식 가능
스캔본 PDF나 이미지 기반 자료는 OCR 기능이 필요함
학습 기록이 장기적으로 사용자별 관리되지는 않음
