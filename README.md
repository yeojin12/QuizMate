# QuizMate

QuizMate는 강의자료를 바탕으로 AI와 대화형 복습 퀴즈를 진행할 수 있도록 프롬프트를 자동 생성해주는 웹 프로토타입입니다.

## 개발 목적

시험 공부 과정에서 학생들은 강의 내용을 읽기만 해서는 자신이 제대로 이해했는지 확인하기 어렵습니다.  
또한 복습 문제를 직접 만들기에는 시간이 오래 걸리고, 강의자료를 매번 정리해서 입력하는 과정도 번거롭습니다.

QuizMate는 이러한 문제를 해결하기 위해 개발한 AI 활용 학습 보조 도구입니다.  
사용자가 강의자료 PDF 또는 텍스트를 입력하면, 선택한 난이도와 문제 유형에 맞춰 GPT에 입력할 수 있는 대화형 복습 퀴즈 프롬프트를 생성합니다.

## 주요 기능

- 강의자료 PDF 업로드
- txt, md 파일 업로드
- 강의 내용 직접 입력
- 문제 난이도 선택
- 문제 수 설정
- 문제 유형 선택
  - 객관식
  - O/X
  - 단답형
  - 서술형
- 대화형 퀴즈 프롬프트 자동 생성
- 생성된 프롬프트 복사

## 사용 방법

1. 강의자료 PDF 파일을 업로드하거나 강의 내용을 직접 입력합니다.
2. 원하는 난이도, 문제 수, 문제 유형을 선택합니다.
3. `대화형 퀴즈 프롬프트 생성하기` 버튼을 누릅니다.
4. 생성된 프롬프트를 복사합니다.
5. GPT에 붙여넣고 대화형 복습 퀴즈를 진행합니다.

## 개선 과정

### 1차 버전

초기 버전은 사용자가 강의 내용을 직접 텍스트로 입력하면 복습용 퀴즈 프롬프트를 생성하는 방식이었습니다.

하지만 실제 사용 과정에서 강의자료는 대부분 PDF 파일로 제공되기 때문에, 사용자가 자료 내용을 직접 복사하거나 따로 정리해야 하는 불편함이 있었습니다.

### 2차 버전

사용자의 입력 부담을 줄이기 위해 PDF 파일 업로드 기능을 추가했습니다.  
이를 통해 사용자는 강의자료를 직접 정리하지 않고도 파일을 업로드하여 퀴즈 생성에 활용할 수 있습니다.

### 3차 버전

처음 생성된 프롬프트는 문제, 정답, 해설을 한 번에 출력하도록 구성되어 있었습니다.  
하지만 이 방식은 사용자가 문제를 풀기도 전에 정답을 보게 되어 실제 복습 효과가 떨어지는 문제가 있었습니다.

이를 개선하기 위해 AI가 한 문제씩 출제하고, 사용자가 답을 입력하면 정답 여부를 판단한 뒤 해설을 제공하는 대화형 퀴즈 방식으로 수정했습니다.

## 한계점

- PDF 파일은 텍스트가 선택되는 자료만 인식할 수 있습니다.
- 스캔본 PDF나 이미지 기반 자료는 별도의 OCR 기능이 필요합니다.
- 현재 버전은 GPT API를 직접 연동하지 않고, 프롬프트를 생성한 뒤 사용자가 GPT에 복사하여 활용하는 방식입니다.

## 향후 개선 방향

- GPT API 연동을 통한 웹페이지 내부 퀴즈 진행
- 사용자의 오답 기록 저장
- 부족한 개념 자동 요약
- 시험 직전 모드 추가
- 오답 개념 기반 추가 문제 생성 기능 추가
