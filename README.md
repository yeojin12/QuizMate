# QuizMate

QuizMate는 강의자료를 바탕으로 AI 학습 정리본과 대화형 복습 퀴즈 프롬프트를 자동 생성해주는 웹 프로토타입입니다.

## 개발 목적

시험 공부 과정에서 학생들은 강의 내용을 읽기만 해서는 자신이 제대로 이해했는지 확인하기 어렵습니다.  
또한 복습 문제를 직접 만들기에는 시간이 오래 걸리고, 강의자료를 매번 정리해서 입력하는 과정도 번거롭습니다.

QuizMate는 이러한 문제를 해결하기 위해 개발한 AI 활용 학습 보조 도구입니다.  
사용자가 강의자료 PDF 또는 텍스트를 입력하면, 선택한 난이도와 문제 유형에 맞춰 GPT에 입력할 수 있는 학습 프롬프트를 생성합니다.

## 주요 기능

- 강의자료 PDF 업로드
- txt, md 파일 업로드
- 강의 내용 직접 입력
- 강의자료 기반 PDF 학습 정리본 생성 요청
- 문제 난이도 선택
- 문제 수 설정
- 문제 유형 선택
  - 객관식
  - O/X
  - 단답형
  - 서술형
- 대화형 퀴즈 프롬프트 자동 생성
- 정답을 바로 보여주지 않는 퀴즈 진행 방식
- 사용자의 답변 이후 정답/오답 판단 및 해설 제공
- `정리` 명령어를 통한 오답노트 PDF 생성 요청
- 틀린 문제만 따로 복습할 수 있는 오답 정리 기능
- 생성된 프롬프트 복사 기능

## 사용 방법

1. 강의자료 PDF 파일을 업로드하거나 강의 내용을 직접 입력합니다.
2. 학습 흐름을 선택합니다.
   - PDF 정리본 먼저 만들기 → 퀴즈 풀기
   - 바로 퀴즈 풀기
3. 원하는 난이도, 문제 수, 문제 유형을 선택합니다.
4. `학습 프롬프트 생성하기` 버튼을 누릅니다.
5. 생성된 프롬프트를 복사합니다.
6. GPT에 붙여넣고 학습을 진행합니다.

## GPT에서 사용할 수 있는 명령어

- `시작`: PDF 정리본을 확인한 뒤 퀴즈를 시작합니다.
- `힌트`: 정답을 직접 말하지 않고 힌트만 받습니다.
- `정리`: 지금까지 푼 문제, 오답, 핵심 개념을 복습용 PDF로 생성합니다.
- `그만`: 퀴즈를 중단하고 현재까지의 결과를 요약합니다.
- `다시`: 방금 문제와 비슷한 유형의 문제를 하나 더 풉니다.

## 개선 과정

### 1차 버전

초기 버전은 사용자가 강의 내용을 직접 텍스트로 입력하면 복습용 퀴즈 프롬프트를 생성하는 방식이었습니다.

하지만 실제 사용 과정에서 강의자료는 대부분 PDF 파일로 제공되기 때문에, 사용자가 자료 내용을 직접 복사하거나 따로 정리해야 하는 불편함이 있었습니다.

### 2차 버전

사용자의 입력 부담을 줄이기 위해 PDF 파일 업로드 기능을 추가했습니다.  
이를 통해 사용자는 강의자료를 직접 정리하지 않고도 파일을 업로드하여 퀴즈 생성에 활용할 수 있습니다.

### 3차 버전

처음 생성된 프롬프트는 문제, 정답, 해설을 한 번에 출력하도록 구성되어 있었습니다.  
하지만 이 방식은 사용자가 문제를 풀기도 전에 정답을 보게 되어 실제 복습 효과가 떨어지는 문제가 있었습니다.

이를 개선하기 위해 AI가 한 문제씩 출제하고, 사용자가 답을 입력하면 정답 여부를 판단한 뒤 해설을 제공하는 대화형 퀴즈 방식으로 수정했습니다.

### 4차 버전

퀴즈를 풀기 전에 강의 내용을 먼저 이해할 수 있도록, 강의자료 기반 PDF 학습 정리본 생성 요청 기능을 추가했습니다.  
또한 사용자가 퀴즈를 중단하기 전 `정리`라고 입력하면, 지금까지 푼 문제와 오답, 중요한 개념을 복습용 PDF로 생성하도록 프롬프트를 개선했습니다.

## 한계점

- PDF 파일은 텍스트가 선택되는 자료만 인식할 수 있습니다.
- 스캔본 PDF나 이미지 기반 자료는 별도의 OCR 기능이 필요합니다.
- 현재 버전은 GPT API를 직접 연동하지 않고, 프롬프트를 생성한 뒤 사용자가 GPT에 복사하여 활용하는 방식입니다.
- QuizMate 웹페이지 자체가 GPT 대화 기록을 직접 불러오지는 못합니다.
- PDF 파일 생성 가능 여부는 GPT 사용 환경에 따라 달라질 수 있습니다.
- PDF 생성이 불가능한 환경에서는 PDF 변환용 문서 형식으로 정리본이 제공됩니다.

## 향후 개선 방향

- GPT API 연동을 통한 웹페이지 내부 퀴즈 진행
- 사용자의 오답 기록 저장
- 웹페이지 내 실제 PDF 다운로드 기능
- 오답 개념 기반 추가 문제 생성 기능
- 시험 직전 모드 추가
- 사용자별 학습 기록 관리 기능
