강의자료를 업로드하면 PDF 정리본 생성, 대화형 퀴즈 진행, 결과 분석용 PDF 생성을 요청하는 프롬프트를 만들어줍니다.

1. 강의자료 입력

PDF 파일을 업로드하거나, 강의 내용을 직접 입력하세요.

txt, md, pdf 파일을 업로드할 수 있습니다. PDF는 글자가 선택되는 텍스트형 PDF만 인식됩니다.

2. 학습 방식 설정

먼저 PDF 정리본을 만들고 공부한 뒤 퀴즈를 풀지, 바로 퀴즈를 풀지 선택하세요.

사용 가능한 명령어

  • 시작: PDF 정리본을 확인한 뒤 퀴즈를 시작합니다.
  • 힌트: 정답을 직접 말하지 않고 힌트만 받습니다.
  • 정리: 지금까지 푼 문제와 오답을 PDF로 정리합니다.
  • 그만: 퀴즈를 중단하고 현재까지의 결과를 요약합니다.
  • 다시: 방금 문제와 비슷한 문제를 하나 더 풉니다.

3. 퀴즈 조건 설정

원하는 난이도, 문제 수, 문제 유형을 선택하세요.
